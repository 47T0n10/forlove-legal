# Privacy Policy

**Last updated: March 22, 2026**

## 1. Overview
ForLove Publisher is a personal desktop tool that publishes video content to TikTok. It does not collect, store, or share personal data from any users other than its owner.

## 2. Data Collected
The only data this application accesses is:
- The owner's TikTok OAuth access token, stored locally on the owner's personal device
- Video files and metadata created by the owner, stored locally on the owner's personal device

## 3. Third-Party Services
This tool communicates with the TikTok API to publish content. TikTok's privacy policy applies to that interaction: [tiktok.com/legal/privacy-policy](https://www.tiktok.com/legal/privacy-policy).

## 4. Data Sharing
No data is shared with any third parties beyond TikTok's API as required to publish content.

## 5. Contact
For any questions, contact the app owner directly.
